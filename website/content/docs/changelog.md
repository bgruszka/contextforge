## v0.1.0 (Initial Release)

### Features
- Sidecar injection via mutating admission webhook
- HTTP header propagation through service chains
- Pod annotations for configuration (`ctxforge.io/enabled`, `ctxforge.io/headers`)
- Lightweight proxy with <5ms latency overhead
- Health checks and graceful shutdown
- Helm chart for easy installation
